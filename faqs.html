Frequently Asked Questions

How do I set up WAP Browser?

There are two different ways you can set up your WAP Browser:
  1. Browsing by GPRS Connection

    • Go to Services
    • Select WAP
    • Go to Settings
    • Select Profiles
    • Select which SIM Card to be used
    • Select your Network, then Activate profile, and wait for Activation.
    • Click Back, then Back
    • Go to Select SIM
    • Select SIM which you configured, wait for Activation
    • Then Click Back
    • Select Hompage, Wait for the Homepage to add load your desired network
    • You have niw successfully configured your WAP browser, Facebook, Twitter, and Yahoo using GPRS connection.

  2. Browsing by Wi-Fi Connection

    • Go to Services
    • Select WAP
    • Go to Settings
    • Select Profiles
    • Select which SIM Card to be used
    • Selece WiFi, then Activate profile, and wait for Activation
    • Click Back, then Back
    • Go to Select SIM
    • Select SIM which you configured, wait for Activation
    • Then Click Back
    • Select Hompage, Wait for the Homepage to add load your desired network
    • You have niw successfully configured your WAP browser, Facebook, Twitter, and Yahoo using GPRS connection.

How do I connect Wi-Fi from my Android device?

  1. Select "Settings"
  2. Then choose "Wireless & Networks
  3. Select "Wi-Fi" to turn WiFI connection ON
  4. Click on "WiFi settings" to scan for available WiFi networks
  5. Select the Wifi network that you want to connect with
  6. Choose "Connect" Note: Connecting to secured networks required password
  7. If password is required, enter the password then "Connect"

How do I set up GPRS in my Android Device?

Go to Menu > Settings > Wireless Networks > Check Data Enabled > Choose your preferred APN
How to Add APN
Menu > Settings > Wireless & Networks > Mobile Networks > Click APN > Press the Left Soft Key > Choose Add APN
For SMART
  • Name- Smart Internet
  • APN-internet
  • Press the left soft key > Click save > GPRS connection can now be used.

What is the correct procedure of charging a battery?

A new battery should be charged for about 3-4 hours. For the first 3-4 times this makes the battery seasoned for better results and longer life Battery should be charged only when it gets fully discharged since all the batteries are designed for certain charge cycles which should not be wasted by semi charging. Also half charged battery when put for charge, start showing false charging quickly even when the battery has not charged fully. When such batteries are put to use, they get discharged also very fast. After the initial charging as above one should follow the normal charging procedure as displayed on the handset or given in the handset manual

What is PIN number and what to do if SIM gets blocked?

As you insert a SIM card in any handset, it asks for PIN (Personal Identification Number) which is a password that is known to the Authorized user only.
The PIN code can be set to protect your SIM card from unauthorized use.
To prevent continuous attempts to 'guess' the PIN number the SIM card automatically gets locked-up after three consecutive wrong entries. If the SIM card is locked-up, the 'Blocked' message will appear and an unlocking code qill be needed. Further repeat wrong entries of PIN may lead to damage of card.
The blocked SIM can be then opened through PUK number (PIN Unlocking Code) which can be obtained from your service provider after establishing your identity. The PUK is an 8-digit code available with your cellular service provider. Also one should remember that 10 time's successive wrong entries of PUK number may damage the SIM card forever.
